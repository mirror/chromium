This folder holds binary text files of recorded events that are to be replayed
within ui/views tests.

For more information about this tool please reference the design document:
https://docs.google.com/a/google.com/document/d/1a7Zcm9ACz4p3D80uBiFcy6owti7hsf1DNyR77cVXx_o/edit?usp=sharing

To turn on the tool run with flag --event-recorder-tool. To start recording
events use Ctrl + Shirt + Z. To stop recording events execute the same command
a second time.
