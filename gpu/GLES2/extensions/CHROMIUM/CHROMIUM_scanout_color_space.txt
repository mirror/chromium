Dependencies

    This extension is written against the OpenGL ES 2.0 Specification.

Overview

    This allows clients to specify the color space used for scanout from
    textures.

New Tokens


New Procedures and Functions

    The command

        void SetColorSpaceForScanout(GLuint texture_id, ColorSpace color_space)

    specifies the color space used for scanout from the given texture.

Errors

    A context lost will result when this call fails, either because of an
    invalid parameter value or because of a runtime error such as an out of
    memory condition.
