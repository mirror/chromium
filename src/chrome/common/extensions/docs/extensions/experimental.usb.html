closeDevice

chrome.experimental.usb.closeDevice(experimental.usb.Device device)

Closes an open device instance. Invoking operations on a device after it has been closed is a safe operation, but causes no action to be taken.

Parameters

device
( experimental.usb.Device )
The device to close.

findDevice

chrome.experimental.usb.findDevice(integer vendorId, integer productId, experimental.usb.DeviceOptions options, function FindDeviceCallback)

Finds the first instance of the USB device specified by the vendorId/ productId pair and, if permissions allow, opens it for use. Upon successfully opening a device the callback is invoked with a populated Device object. On failure, the callback is invoked with null.

Parameters

vendorId
( integer )
The vendor ID of the USB device to find.
productId
( integer )
The product ID of the USB device to find.
options
( experimental.usb.DeviceOptions )
Undocumented.

( function )
Undocumented.

device
( optional experimental.usb.Device )
Undocumented.

experimental.usb.UsbEvent

( object )
Undocumented.
data
( optional ArrayBuffer )
If the transfer was an input transfer then this field will contain all of the input data requested.
error
( optional string )
If resultCode is not 0, this field will contain a textual description of the error that occurred during the transfer.
resultCode
( optional integer )
A value of 0 indicates that the transfer was a success. Other values indicate failure.
type
( string )
A string indicating the type of the event. Currently will only contain the value 'transferResult'.
